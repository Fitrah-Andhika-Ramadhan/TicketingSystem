
================================================================================
                    LANDING PAGE PUBLIC ACCESS - ADMIN PANEL
                              COMPLETE SETUP
================================================================================

OVERVIEW:
Admin sekarang memiliki akses mudah ke landing page public dari berbagai tempat
di admin panel. Admin bisa navigate, preview, dan manage landing page dengan mudah.

================================================================================
1. NAVBAR PREVIEW BUTTON
================================================================================

LOKASI: Top right navbar (di setiap halaman admin)
TAMPILAN: "Preview Landing" button dengan icon ExternalLink
FUNGSI: Buka landing page di tab baru
TARGET: http://localhost:3000/landing

CARA PAKAI:
  1. Login ke dashboard
  2. Lihat navbar di atas
  3. Klik "Preview Landing" (cukup kecil di mobile, gunakan responsive menu)
  4. Landing page terbuka di tab baru

================================================================================
2. LANDING MANAGER PAGE - QUICK ACCESS CARDS
================================================================================

LOKASI: /admin/landing-manager
TERDAPAT: 3 Quick Access Cards di atas form editor

CARD 1 - Preview Landing (Blue)
  - Tombol: "Open Landing Page"
  - Fungsi: Buka landing page di tab baru
  - Target: /landing
  - Deskripsi: "See how your landing page looks"

CARD 2 - Media Gallery (Purple)
  - Tombol: "Go to Media Manager"
  - Fungsi: Navigate ke media manager
  - Target: /admin/media-manager
  - Deskripsi: "Manage images and videos"

CARD 3 - Landing Info (Green)
  - Info: Public URL path (/landing)
  - Info: Siapa bisa akses (everyone, no login)
  - Deskripsi: "Public access details"

================================================================================
3. MEDIA MANAGER PAGE - QUICK ACCESS CARDS
================================================================================

LOKASI: /admin/media-manager
TERDAPAT: 2 Quick Access Cards di atas media list

CARD 1 - Preview Landing (Blue)
  - Tombol: "Open Landing Page"
  - Fungsi: Buka landing page di tab baru
  - Target: /landing
  - Deskripsi: "See how media appears on landing page"

CARD 2 - Manage Content (Green)
  - Tombol: "Go to Landing Manager"
  - Fungsi: Navigate ke landing manager
  - Target: /admin/landing-manager
  - Deskripsi: "Edit hero, stats, and about section"

================================================================================
4. SIDEBAR ADMIN MENU
================================================================================

MENU ITEMS (Admin Only):
  
  1. "Manage Landing"
     - Icon: FileText
     - URL: /admin/landing-manager
     - Akses: SUPER_ADMIN, ADMIN
     - Fungsi: Edit landing page content

  2. "Media Manager"  
     - Icon: Image
     - URL: /admin/media-manager
     - Akses: SUPER_ADMIN, ADMIN
     - Fungsi: Manage images/videos

  3. "Settings"
     - Icon: Settings
     - URL: /admin/settings
     - Akses: SUPER_ADMIN, ADMIN
     - Fungsi: Admin configuration

================================================================================
5. COMPLETE USER FLOW
================================================================================

LOGIN WORKFLOW:
  Login (admin@natagroup.com) 
    → Dashboard (Main page)
    → Navbar has "Preview Landing" button
    → Sidebar has Admin menu

ADMIN TO LANDING FLOW 1 (Quickest):
  Dashboard → Click Navbar "Preview Landing" → Landing page opens in new tab

ADMIN TO LANDING FLOW 2 (With edits):
  Dashboard 
    → Sidebar: "Manage Landing" 
    → Edit hero/stats/about
    → Click blue "Open Landing Page" card
    → See changes in real-time

ADMIN TO LANDING FLOW 3 (Media management):
  Dashboard 
    → Sidebar: "Media Manager"
    → Add/edit images/videos
    → Click blue "Open Landing Page" card
    → See media in gallery

QUICK NAVIGATION:
  Landing Manager → Media Manager (via purple card)
  Media Manager → Landing Manager (via green card)
  Either page → Landing page preview (via blue card)
