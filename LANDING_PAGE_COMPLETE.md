## API ENDPOINTS

### Landing Content API
```
GET /api/landing/content
- Returns: Hero, Stats, About data
- Access: Public (no auth)

PUT /api/landing/content
- Update content
- Access: Admin only
```

### Landing Media API
```
GET /api/landing/media
- Returns: All media items
- Access: Public (no auth)

POST /api/landing/media
- Add new media
- Access: Admin only

PUT /api/landing/media/:id
- Update media
- Access: Admin only

DELETE /api/landing/media/:id
- Delete media
- Access: Admin only
```
